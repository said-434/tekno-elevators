# موقع TEKNO_ELEVATORS

موقع جاهز لشركة المصاعد TEKNO_ELEVATORS، يشمل:
- `index.html` الصفحة الرئيسية
- `about.html` صفحة نبذة عن الشركة
- `services.html` صفحة الخدمات
- `elevators.html` صفحة لإضافة المصاعد بنفسك
- `explain.html` صفحة شرح مع صور
- `contact.html` صفحة التواصل
- `styles.css` تنسيق التصميم
- `script.js` تفاعل النماذج

## تشغيل الموقع محلياً
1. افتح المجلد `New folder`.
2. انقر مزدوجاً على `index.html` لفتحها في المتصفح.

## تنظيم الصور والشعار
1. أنشئ مجلداً جديداً اسمه `images` داخل المجلد.
2. ضع الشعار الحقيقي في `images/logo.png` أو `images/logo.jpg`.
3. ضع الصور الحقيقية التالية في المجلد بنفس الأسماء:
   - `images/elevator-1.jpg`
   - `images/elevator-2.jpg`
   - `images/elevator-3.jpg`
   - `images/elevator-4.jpg`
   - `images/elevator-5.jpg`
   - `images/elevator-6.jpg`

مهم: اللوجو يجب أن يكون منفصلًا في `images/logo.jpg` أو `images/logo.png`، والصور الستة هي صور المعرض في صفحة `explain.html`.
إذا كانت الصور بصيغة PNG، يمكنك تغيير الامتداد إلى `.png` مع الحفاظ على نفس الاسم الأساسي. الموقع سيعرض الصور الحقيقية تلقائيًا إذا كانت موجودة.

## إرسال الرسائل مباشرة من النموذج
- صفحة `contact.html` الآن ترسل الرسائل مباشرة إلى بريد `Tecno.elevators.sa@gmail.com` باستخدام خدمة Formsubmit.
- عند الضغط على زر "أرسل الرسالة" يتم توجيه النموذج تلقائيًا للخدمة ثم إرسال الرسالة إلى البريد المحدد.
- لتغيير البريد استخدم عنوان البريد المطلوب في قيمة `action` داخل نموذج `contact.html`.

## رفع الموقع على الإنترنت بسهولة
### GitHub Pages
1. أنشئ حساب GitHub.
2. أنشئ مستودع جديد ورفع محتويات المجلد.
3. فعل GitHub Pages من إعدادات المستودع واختر الفرع `main`.
4. سيظهر الموقع عبر رابط GitHub Pages.

### Netlify
1. افتح `https://app.netlify.com/drop`.
2. اسحب المجلد `New folder` أو الملفات والملف `images`.
3. سيعطيك رابط مباشر للموقع.

## إضافة الموقع إلى Google
1. افتح `https://search.google.com/search-console`.
2. أضف رابط الموقع بعد نشره.
3. اختَر طريقة التحقق (Google يمنح أكثر من طريقة).
4. بعد التحقق، اطلب فهرسة الصفحة عبر "URL Inspection".

> بعد نشر الموقع، قد يأخذ ظهور الموقع في نتائج البحث بعض الوقت.
